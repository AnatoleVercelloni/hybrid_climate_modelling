

HR -> 21 600 colums ~40 T
LR -> 384    colums ~750G

10 simulated years -> one sample every 20 minutes
==> 10*365*72*2 = 525 600 files (4320 files/month)


E3SM-MMF.ml'X'.'yyyy'-'mm'-'dd'-'sssss'.nc with
	X = i or o
	0001  <= yyyy  <= 0010
	01    <=  mm   <= 12
	01    <=  dd   <= 31
	00000 <= sssss <= 86400
	
	
	
	
For input files:

	dimensions:
		ncol = 21 600 (resp. 384) ;
		lev = 60 ;
	28 variables:
		int ymd ;                       |      -|10201 <= year-month-day <= 101231 
		int tod ;                       |      -|0 <= time of day <= 86 400
		double cam_in_ALDIF(ncol) ;     |      -|Albedo for diffuse longwave radiation

		double cam_in_OCNFRAC(ncol) ;   |      -|?? Ocean area fraction ??
		double cam_in_SNOWHICE(ncol) ;  |      m|Snow depth over ice

		double pbuf_COSZRS(ncol) ;      |      -|Cosine of solar zenith angle

========================PREPROCESSING======================

(i)   Downsampling in time (1/7)
(ii)  collapse horizontally location and time into a single dimension
(iii) normalize variables , with these statistics calculated separately at each of the 60 vertical levels for the four variables with vertical dependence
(vi)  concatenate variables into multi-variate input and output vectors for each sample
